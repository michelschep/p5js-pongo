# Pongo 🎈

**Pong-style arcade balloon game with an Onda, Spain theme!**

A *Pang/Bubble Trouble* clone built with [p5.js](https://p5js.org/).

## 🎮 Gameplay

- Move left/right with **← →** or **A/D**
- Shoot upward with **Space** or **↑**
- Hit balloons to split them: Large → 2 Medium → 2 Small → gone
- Score points for each balloon popped
- 3 lives — avoid touching balloons!
- Complete a level by clearing all balloons

## 🏡 Theme: Onda, Castellón

Onda is a municipality in Castellón, Spain, famous worldwide for its ceramic tiles (azulejos). The game features:
- Mediterranean blue sky
- White Andalusian-style buildings
- Hand-drawn ceramic tile floor
- Spanish-style character with sombrero

## 🕹️ Controls

| Action  | Keyboard         | Mobile      |
|---------|-----------------|-------------|
| Move    | ← → / A D       | ◀ ▶ buttons |
| Shoot   | Space / ↑       | 🔫 button   |

## 🔊 Sound Effects

Web Audio API — no external dependencies:
- Shoot, Pop, Bounce, Death, Level Complete, Score

## 🛠️ Tech Stack

- p5.js v1.7.0 (CDN)
- Vanilla JavaScript
- Web Audio API
- Mobile-responsive with touch controls
